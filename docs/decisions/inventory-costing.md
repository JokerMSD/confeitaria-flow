# Decision: Inventory Costing

## Decisao Atual
- Custo de ingrediente deve tender a media ponderada das compras.
- Compras podem informar valor pago real.
- Itens comprados em `un` ou `caixa` podem informar rendimento real em unidade de receita.

## Motivo
- Preco de mercado varia.
- Custo fixo isolado envelhece rapido.
- Media ponderada reflete melhor o custo corrente.

## Efeitos
- Receitas usam custo mais realista.
- Guia de compra estima gasto com base melhor.
- Catalogo e preco ideal ficam mais coerentes.
